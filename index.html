Hate the dull feel of a regular terminal?

Try TerminalChad

Features

    - Customisable Themes

    - Themes Library

    - PackageManager(like) Themes Usage

TerminalChad is an extension for Windows Terminal which allows for the easy sharing of Themes. It also allows for the easy switching of themes.
